Para correr el programa de "estadistica.py" en donde se encuentran todos las tablas y datos estadísticos,
una vez que la persona se encuentre en el repositorio o carpeta con dicho archivo, ejecutar el comando:
"python3 estadistica.py" en la terminal o línea de comandos.

También, si se tienen instaladas las librerias necesarias (pandas, notebook, seaborn), se puede ejecutar celda por celda en el editor
de codigo que lo permita (preferiblemente gitpod) las secciones de código para vizualizar lo que realizan
parte por parte.

Para la sección de gráficas, cada gráfica está comentada para evitar
amontonamiento de las gráficas en una sola. Se puede ir "descomentando" cada línea para generar las gráficas que realizamos.
